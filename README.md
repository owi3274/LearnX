# LearnX
Course Cards Project

Overview

This project displays a collection of course cards with images, descriptions, and pricing details. It features a modern, responsive design using HTML and CSS.

Features

Beautiful, responsive course cards.

Dynamic sale badges and action buttons.

Smooth scrolling for horizontal card navigation.

Gradient button effects.

Custom-styled scrollbar.

Technologies Used

HTML5

CSS3

Google Fonts (Montserrat)

File Structure

project-folder/
│── index.html        # Main HTML file
│── styles.css        # CSS file for styling
│── images/           # Folder containing course images
│── README.md         # Project documentation

Setup Instructions

Clone or download the repository.

Ensure all images are correctly placed in the images/ folder.

Open index.html in your browser to view the project.

How to Use

Scroll horizontally to navigate through courses.

Hover over action buttons for animations.

View sale badges on discounted courses.

Troubleshooting

Images Not Showing?

Verify that image paths in index.html match the filenames in the images/ folder.

Ensure the object-fit: cover; CSS property is not hiding the images.

Check the browser console (F12) for missing file errors.
